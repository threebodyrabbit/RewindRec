# RewindRec

An open source project that helps you discover and revisit older entries in a franchise before a new release drops.

## What It Does

When a new installment of a franchise (game, movie, TV show, book series, etc.) is announced or about to release, RewindRec recommends which older titles in that franchise you should experience first — so you're fully caught up and ready.

## Features

- Detects upcoming franchise releases
- Recommends older titles to watch, play, or read beforehand
- Ranks recommendations by relevance (story order, lore importance, etc.)
- Supports multiple media types: games, films, TV shows, books
